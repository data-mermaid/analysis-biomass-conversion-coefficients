# MERMAID - Changing your biomass conversion coefficients

Example showing how biomass conversion coefficients can be changed for your MERMAID fish data - either using a previous version of the MERMAID reference set or a custom table of coefficients.

Rendered file can be found [here](https://data-mermaid.github.io/analysis-biomass-conversion-coefficients/biomass_conversion_coefficients.html).
